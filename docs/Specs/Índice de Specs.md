## Como funciona o fluxo

```
/spec [requisito]   → gera spec com gate humano
/plan [spec]        → cria plano técnico após aprovação
/front ou /back     → implementação
/review             → revisão em dois estágios
```

Os arquivos de spec ficam em `.ai-core/specs/` e os planos em `.ai-core/plans/`.
