# Personas

## Recrutador

**Quem é:** Recrutador técnico ou gestor de engenharia buscando contratar um engenheiro de software sênior.

**Objetivo:** Avaliar experiência, projetos e fit técnico de Roberto Nicoletti de forma rápida e convincente.

**Dor principal:** Dificuldade em encontrar evidências concretas de senioridade e qualidade de entrega em um único lugar. Currículos falam de cargos; GitHub mostra código bruto; portfólios mostram screenshots.

**Nível técnico:** Técnico a semi-técnico.

**Jornada:**
1. Acessa `nico.dev` via LinkedIn, indicação ou busca
2. Navega pelo portfólio de projetos
3. Consulta o currículo para histórico profissional
4. Acessa subprojetos via subdomínios para ver aplicações reais rodando
5. Usa o formulário de contato para iniciar conversa

**O que precisa ver:**
- Projetos reais e funcionando, não só screenshots
- Stack técnica clara e moderna
- Histórico profissional organizado e legível
- Canal de contato direto e fácil

## Desenvolvedor

**Quem é:** Desenvolvedor da comunidade que chega via busca orgânica, redes sociais ou indicação.

**Objetivo:** Consumir conteúdo técnico e explorar projetos open source.

**Dor principal:** Conteúdo técnico de qualidade é escasso; quer aprender com quem está no mercado.

**Nível técnico:** Técnico.

**Jornada:**
1. Chega via busca orgânica ou link compartilhado
2. Lê artigos técnicos no blog
3. Explora repositórios linkados nos projetos

**O que precisa ver:**
- Blog com artigos de qualidade e profundidade
- Links para repositórios GitHub dos projetos
- Conteúdo atualizado e relevante
