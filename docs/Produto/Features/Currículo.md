# Feature: Currículo

**Status:** 🔄 Em andamento

## O que é

Página de CV com experiência profissional, habilidades técnicas e histórico de formação. Voltada principalmente para [[Personas#Recrutador|recrutadores e empresas]].

## Spec relacionada

- [[Índice de Specs#2026-05-16 Página de Currículo]]

## Seções previstas

- Resumo profissional
- Experiência profissional (empresas, cargos, período, realizações)
- Stack técnica / Skills
- Formação acadêmica
- Links (GitHub, LinkedIn, projetos)

## Considerações de UX

- Deve ser **imprimível** — versão PDF ou estilo de impressão via CSS
- Leitura rápida — recrutador lê em 30 segundos
- Mobile-first (recrutadores frequentemente acessam no celular)
