# Visão do Produto

## O que é

**Nico.dev** é o site pessoal e portfólio profissional de Roberto Nicoletti — engenheiro de software sênior. O projeto vai além de um portfólio estático: é um monorepo Turborepo que reúne o site principal e subprojetos independentes com deploy em subdomínios próprios (`projeto.nico.dev`), mostrando produtos **reais em funcionamento** — não apenas screenshots.

## Tagline

> *Engenheiro de software construindo produtos digitais escaláveis*

## Estágio

**MVP** — site principal em desenvolvimento ativo.

---

## Problema que resolve

Recrutadores e empresas têm dificuldade em encontrar evidências concretas de senioridade e qualidade de entrega de um engenheiro em um único lugar. LinkedIn mostra cargos, GitHub mostra código, mas nenhum dos dois mostra o produto funcionando com produto real.

## Solução

Um hub centralizado em `nico.dev` onde:
- Projetos são apresentados com contexto, tecnologias e links para o produto rodando
- Subprojetos são aplicações reais acessíveis em subdomínios
- O currículo está integrado, estruturado e fácil de consumir
- O contato é direto, sem intermediários

## Diferenciais

- **Subprojetos ao vivo** — não é portfólio de screenshots; o produto pode ser testado
- **Monorepo com shared packages** — demonstra maturidade técnica na própria estrutura do projeto
- **Conteúdo técnico via blog** — posiciona como referência, não só candidato
