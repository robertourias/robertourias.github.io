# robertourias.github.io

Meu portfólio pessoal desenvolvido com Next.js.

## Como executar localmente

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

3. Acesse no navegador: http://localhost:3000

## Tecnologias

- Next.js
- React
- TypeScript
- Tailwind CSS
